# I call the alphabet game

## Instructions
------------

Write a program that prints the alphabet in lowercase and uppercase, followed by a new line.

use different functions to print the same alphabet printf, puts, etc…

Examples:

- TEST with for : abcdefghijklmnopqrstuvwxyz
- TEST with for ASCII : abcdefghijklmnopqrstuvwxyz
- TEST with while : abcdefghijklmnopqrstuvwxyz
- TEST with while ASCII : abcdefghijklmnopqrstuvwxyz
